# 🎓 Exam Eligibility System

This is a desktop application developed for the University of Vavuniya to manage student exam eligibility.

## 🔧 Technologies Used
- C#
- Windows Forms
- LINQ to SQL
- SQL Server

## 🚀 Features
- Student registration
- Subject management
- Marks management
- Eligibility checking system

## 💡 Purpose
To automate and simplify the process of determining student eligibility for exams.
